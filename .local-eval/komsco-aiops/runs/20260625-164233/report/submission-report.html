  • 총 응답 시간: /api/chat 요청 시작부터 SSE stream 종료까지

  • 추정 tok/s: estimatedTokens / totalSeconds

점수 산식

  • 시나리오별 필수 키워드, 구조화 여부, 한국어 포함 여부, 응답 길이 검사

  • 운영 단순성은 runtime 종류, secondary store, switch 안정성을 반영
